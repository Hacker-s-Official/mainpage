# MAINPAGE
LIBERLUNAのメインページです。
## Build
[Deno](https://deno.land)を使い、ビルドします。
```shell
deno run -A build.ts
```
`./src`フォルダの中身が`./dist`フォルダにビルドされます。
### Deploy
GitHubで`master`ブランチからリリースを行うと、自動的に[GitHub pages](https://liberluna.github.io/mainpage/)にdeployされます。
